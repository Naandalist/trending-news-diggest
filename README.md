# 📰 Trending News Diggest 🚀

Welcome to **Trending News Diggest** – a Bash-powered news-digesting machine that fetches, processes, and summarizes trending news, all with a side of satire and fun! 🎉

## 📚 What is Trending News Diggest?

This project is a pure Bash automation script that:
- Fetches the latest news data from your favorite sources
- Converts it to beautiful Markdown
- Summons an AI to deliver sharp, hilarious, and insightful Indonesian summaries
- Organizes everything in neat, date-stamped folders
- Pushes the magic straight to your Git repository

**No Node.js, no Express, just shell script wizardry! 🧙‍♂️**

## ✨ Features

- 🌍 **Curated Trending News:** Get the latest, most buzzworthy headlines, fresh from the source.
- 💾 **Markdown Magic:** News is saved as clean, readable Markdown files for your delight.
- 🤖 **AI Commentary:** Each article comes with an AI-generated, satirical Indonesian summary—sometimes a little too honest!
- 🛠️ **Easy Automation:** Just run the script and watch the magic happen.
- 🚀 **Git Integration:** Commits and pushes news updates for you.

## 🚦 Prerequisites

You'll need these command-line tools installed:
- `curl`
- `jq`
- `pandoc`
- `git`

Just run the script and it'll check for you. If you’re missing something, it’ll tell you!

## ⚡ Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Naandalist/trending-news-diggest.git
   cd trending-news-diggest
   ```

2. **Set up your `.env` file:**
   - Required variables: `URL_HOME`, `URL_DETAIL`, `AI_ENDPOINT`, `USER_AGENT`, `HOST_HEADER`
   - (Optional) `ARCHIVE_BASE` to customize your archive folder location.

   *Need a sample `.env` file?*  
   **Just contact me—I'm super easy to reach!**

3. **Run the script:**
   ```bash
   ./your-script-name.sh
   ```

4. **Enjoy your fresh, AI-powered Markdown news in the `archive/` directory!**
